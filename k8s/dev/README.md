# Kubernetes Manifests - dev

This folder contains Day 2 runtime manifests for AKS dev.

## Files

- 00-namespace.yaml
- 01-configmap-backend.yaml
- 01-configmap-backend-nginx.yaml
- 01-secret-backend.template.yaml
- 10-deployment-backend.yaml
- 11-service-backend.yaml
- 20-deployment-frontend.yaml
- 21-service-frontend.yaml
- 30-ingress.yaml
- 40-job-migrate.yaml

## Apply order

1. Namespace
2. ConfigMaps
3. Secret (generated by pipeline from Key Vault values)
4. Deployments and Services
5. Ingress
6. Migration Job (triggered in pipeline, not on app startup)

## Local validation

```powershell
kubectl apply --dry-run=client -f k8s/dev/
```

## Pipeline replacement contract

Replace IMAGE_TAG in these files before apply:
- 10-deployment-backend.yaml
- 20-deployment-frontend.yaml
- 40-job-migrate.yaml

Expected replacement value:
- IMAGE_TAG -> commit SHA (or release tag)

## Runtime injection model

- Non-sensitive config: ConfigMap (backend-config)
- Sensitive values: Secret (backend-secrets)
